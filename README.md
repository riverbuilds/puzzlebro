# puzzlebro

The first thing I ever built with HTML and CSS. Six little puzzles, no real theme — just random stuff I thought was fun (or annoying) to figure out while I was learning the basics.

[play it here](https://riverbuilds.github.io/puzzlebro/)

Nothing serious, just wanted to see if I could build something that actually worked while I was still figuring out what a `<div>` was. Some of the puzzles are dumb on purpose. No judging.

Built with plain HTML + CSS. No frameworks, no libraries, no build tools — just files.
